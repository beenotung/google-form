# google-form

Extract Google Form questions to JSON.

## Usage

1. Open the Google Form edit page in your browser.
2. Open the Developer Tools.
3. Copy and paste the `userscript.js` code into the Developer Tools console.
4. Press Enter to run the code.
5. The JSON data will be printed to the console. It can be saved for later use.
